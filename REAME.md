# SQL Practice Repository

## Overview

This repository contains SQL practice scripts focused on learning and understanding core database concepts and query writing.  
It includes beginner-to-intermediate level SQL programs covering:

- DDL Commands
- DML Commands
- TCL Commands
- DCL Commands
- DQL Queries
- Joins
- Subqueries
- Functions
- Analytical Functions
- Mock Questions & Assignments

The repository was created as a personal learning workspace for improving SQL query writing and database management skills.
